Vocabulary 0.1.7-SNAPSHOT

Released under the Eclipse Public License

Utilities to map between namespaced keywords and URIs, and support language-tagged strings.

Installation

To install, add the following dependency to your project or build file:

[ont-app/vocabulary "0.1.7-SNAPSHOT"]
